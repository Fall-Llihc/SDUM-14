Sains Data Untuk Masyarakat

Workshop Praktis Penguasaan Data Entry dan Spreadsheet Dasar sebagai Kunci Keterampilan Masa Depan Siswa SMK Telkom Bandung

Tentang Program SDUM

Lokasi & Waktu Pelaksanaan

Lokasi : SMK Telkom Bandung, Jl. Radio Palasari Road, Citeureup, Kec. Dayeuhkolot, Kabupaten Bandung, Jawa Barat

Waktu : 16 Desember 2025

Bidang Fokus : Workshop

Deskripsi Program

Program Sains Data untuk Masyarakat (SDUM) adalah inisiatif dari mahasiswa Telkom University untuk memberikan pelatihan praktis kepada siswa SMK dalam bidang pengolahan data. Workshop ini mencakup:

  • Pengenalan konsep data dan pentingnya literasi data
  • Pelatihan Microsoft Excel untuk pengolahan data dasar
  • Teknik data entry yang efektif dan efisien
  • Penggunaan spreadsheet untuk analisis data sederhana
  • Praktik langsung dengan studi kasus nyata

Dampak & Pencapaian

Workshop ini berhasil melatih puluhan siswa SMK Telkom Bandung dengan materi yang aplikatif dan mudah dipahami. Para peserta menunjukkan antusiasme tinggi dan mampu mengaplikasikan keterampilan data entry serta spreadsheet untuk kebutuhan akademis dan persiapan dunia kerja.
